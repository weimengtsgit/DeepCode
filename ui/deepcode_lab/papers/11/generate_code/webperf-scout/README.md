# WebPerf Scout

WebPerf Scout is a comprehensive web performance analysis CLI tool designed to measure and evaluate web page performance across multiple metrics.

## Features

- Automated web performance measurement
- Detailed performance metric collection
- Cross-browser performance analysis
- Customizable reporting

## Installation

```bash
npm install webperf-scout
```

## Usage

```bash
webperf-scout analyze https://example.com
```

## Performance Metrics

- Largest Contentful Paint (LCP)
- First Contentful Paint (FCP)
- Cumulative Layout Shift (CLS)
- Time to Interactive (TTI)
- Total Blocking Time (TBT)
