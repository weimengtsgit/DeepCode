# WebPerf Scout

WebPerf Scout is an automated web performance testing and scoring toolkit.

## Overview

WebPerf Scout provides comprehensive web performance analysis, including:
- URL discovery
- Performance metric collection
- Detailed performance scoring
- Visualization and reporting

## Installation

1. Clone the repository
2. Run `npm install`
3. Configure `.env` file
4. Run `npm run setup`

## Usage

```bash
# Basic usage
npx webperf-scout analyze https://example.com

# Crawl and analyze multiple pages
npx webperf-scout crawl https://example.com --max-depth 2
```

## Features

- Automated web performance testing
- Comprehensive performance scoring
- Multi-browser support
- Detailed HTML and JSON reporting
