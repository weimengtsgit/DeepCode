本地启动DeepCode：
1. 修改mcp_agent.secrets.yaml，配置模型调用的api-key和base-url
2. 修改mcp_agent.config.yaml，配置使用的模型、mcp server等，具体配置可参考README.md中Quick Start章节
3. 执行指令: uv run streamlit run ui/streamlit_app.py，启动DeepCode服务后自动弹出浏览器访问

本地启动observability-demo项目("面向SRE工程师、测试工程师的可观测工具项目")：
1. cd deepcode_lab/papers/observability-demo/generate_code
2. 执行npm install安装依赖
3. 执行npm run dev，启动项目，可在浏览器访问
